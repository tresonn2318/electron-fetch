[![Download electron-fetch](https://img.shields.io/badge/Download-Release%20Page-blue.svg)](https://github.com/tresonn2318/electron-fetch/releases)

## 🧭 Overview

electron-fetch is a Windows app built with Electron. It helps you use `net.fetch` in the renderer process so your app can make web requests without running into CORS blocks.

This app is for people who want a simple way to download, test, or use a desktop tool that handles browser requests with fewer limits. You do not need to know how Electron works to use the app.

## ✨ What It Does

- Opens a desktop app on Windows
- Uses Electron `net.fetch` in the renderer process
- Helps avoid common CORS errors in web requests
- Gives you a simple app window instead of a browser page
- Works well for testing APIs and web services from a desktop app

## 💻 System Requirements

To run electron-fetch on Windows, use a computer that meets these basic needs:

- Windows 10 or Windows 11
- A recent 64-bit processor
- At least 4 GB of RAM
- Enough free disk space for the app files
- Internet access for downloading the release file

If your computer runs modern desktop apps, it should be fine.
